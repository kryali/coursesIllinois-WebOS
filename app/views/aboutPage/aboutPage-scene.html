About Us
coursesIllinois was conceived, designed and created by Kiran Ryali and Atul Gupte, both Juniors at UIUC for the Palm Student Showcase.

It is our solution to provide course information in an organized and systematic format, on the one device thats with you at all times - your mobile phone.

A few currently unmet goals for this project are:
  1. Calendar integration - adding sections to calendar events
  2. Saved courses - keeping track of a user's courses
  3. Task list tagging - tag assignments and todo's with saved course id
  4. Searchable Lists - filter through list with user input


Content used is from the UIUC Course Catalog and www.ratemyprofessors.com
